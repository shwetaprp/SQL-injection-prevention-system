# 🔐 Secure Registration System

## 📌 Overview

A simple Flask-based registration system that protects user data from **SQL Injection attacks** using secure coding techniques.

---

## 🎯 Features

* Secure user registration
* Password hashing (SHA-256)
* SQL Injection protection
* Input validation
* Duplicate user check

---

## 🛠️ Tech Stack

* Python (Flask)
* SQLite
* HTML, CSS

---

## ▶️ Run Project

```bash
pip install flask
python secure_app.py
```

Open in browser:
http://127.0.0.1:5000/
